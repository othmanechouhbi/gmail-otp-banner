# Changelog

All notable changes to Gmail OTP Banner are documented here.

## 1.0.0 - 2026-05-26

- Initial production release structure.
- Added Chrome Manifest V3 extension source under `extension/`.
- Added Gmail OAuth with `chrome.identity.getAuthToken`.
- Added local OTP detection from Gmail readonly API.
- Added multi-account support for up to four Gmail accounts.
- Added baseline logic to prevent old OTPs from displaying after connection.
- Added modern floating banner with copy and close controls.
- Added privacy, limited use, support, security, and OAuth verification documents.
- Added static website pages for public policy hosting.
- Added Chrome Web Store assets and release checklist.
