# Gmail OTP Banner

Gmail OTP Banner is a Chrome extension that helps you copy recent verification codes from Gmail faster.

After you connect your Gmail account with Google, the extension watches for new recent verification emails and displays the newest OTP in a small floating banner with a Copy button.

## Features

- Detects recent Gmail verification codes.
- Shows a floating banner in the top-right corner of the browser.
- Includes a one-click Copy button.
- Supports multiple Gmail accounts.
- Ignores old codes received before you connected the account.
- Uses Google sign-in only; it never asks for your Gmail password.
- Runs OTP detection locally in your browser.

## Install

Install Gmail OTP Banner from the Chrome Web Store.

After installation, pin the extension to your Chrome toolbar for easier access.

## How To Use

1. Click the Gmail OTP Banner icon in Chrome.
2. Click **Connect an account**.
3. Sign in with Google.
4. Allow Gmail read-only access.
5. Receive a new OTP or verification email.
6. The code appears in a floating banner.
7. Click **Copy** to copy the code.

## Privacy

Gmail OTP Banner requests Gmail read-only access so it can scan recent Gmail messages for verification codes.

The extension reads only recent emails needed for OTP detection. Processing happens locally in your browser. Gmail OTP Banner does not store full email bodies, does not sell email content, and does not use email content for advertising.

The extension stores only small local data needed to work, such as connected account information, recent message IDs, ignored message IDs, and the latest detected OTP.
