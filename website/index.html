Gmail OTP Banner

A privacy-focused Chrome extension that detects recent Gmail verification codes locally and shows them in a clean floating banner.

Local processing
Email content is scanned in the browser, not on a remote server.
Minimal Gmail scope
Uses Gmail readonly access only.
No old codes
Codes received before account connection are not shown.

Limited Use Disclosure

Gmail OTP Banner's use and transfer of information received from Google APIs will adhere to the Google API Services User Data Policy, including the Limited Use requirements.
