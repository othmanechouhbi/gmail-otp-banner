Gmail OTP Banner

Gmail OTP Banner is an open-source Chrome extension that connects to Gmail using Google OAuth, detects new verification codes received after connection, and displays the newest valid OTP in a small browser banner.

Current status: Gmail OTP Banner is not available on the Chrome Web Store yet. Download the source code from GitHub and install it manually using Chrome Developer Mode.

What It Does

  • Connects to Gmail using Google OAuth.
  • Scans recent Gmail messages for OTP detection.
  • Shows the newest valid OTP in a small banner.
  • Runs locally in your browser.
  • Does not send Gmail content to external servers.

Manual Installation

  1. Download or clone the repository.
  2. Open chrome://extensions/.
  3. Enable Developer Mode.
  4. Click Load unpacked and select the extension folder.
  5. Copy the generated Chrome Extension ID.
  6. Create a Google Cloud project and enable Gmail API.
  7. Configure the OAuth consent screen and add yourself as a test user.
  8. Create an OAuth Client ID of type Chrome Extension.
  9. Paste the Chrome Extension ID into Google Cloud.
  10. Paste the OAuth Client ID into extension/manifest.json.
  11. Reload the extension and click Connect an account.

Google Cloud Paths

  • Google Cloud Console → APIs & Services → Library → Gmail API → Enable
  • Google Cloud Console → Google Auth Platform → Branding
  • Google Cloud Console → Google Auth Platform → Audience → Test users
  • Google Cloud Console → Google Auth Platform → Clients → Create Client → Chrome Extension
