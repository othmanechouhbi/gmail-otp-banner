Privacy Policy

Effective date: May 26, 2026

Gmail OTP Banner processes Gmail data locally in your browser to find recent verification codes. It does not sell data, display ads, use Gmail content for analytics, or send full email content to external servers.

Google Data Access

The extension requests https://www.googleapis.com/auth/gmail.readonly to search recent Gmail messages for verification-code emails.

Stored Data

The extension stores connected account email, OAuth token, connection timestamp, baseline message timestamp, latest OTP, latest message ID, processed message IDs, and ignored message IDs in Chrome local storage.

Limited Use

Gmail OTP Banner's use and transfer of information received from Google APIs will adhere to the Google API Services User Data Policy, including the Limited Use requirements.
