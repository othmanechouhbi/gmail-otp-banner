Privacy Policy

Effective date: May 26, 2026

Overview

Gmail OTP Banner is an open-source Chrome extension for local installation. It is not currently available on the Chrome Web Store.

Users download the source code, configure their own Google Cloud project, create their own OAuth Client ID, and install the extension manually using Chrome Developer Mode.

Data Processing

Gmail OTP Banner connects to Gmail using Google OAuth and requests Gmail read-only access so it can detect verification codes.

The extension processes Gmail data locally in the browser. It only scans recent Gmail messages for OTP detection and does not store full email bodies.

What We Do Not Do

  • We do not sell data.
  • We do not send Gmail content to external servers.
  • We do not use Gmail content for advertising.
  • We do not store full email bodies.

Local Storage

The extension may store small local data required for functionality, such as connected account information, recent message IDs, ignored message IDs, and the latest detected OTP.
