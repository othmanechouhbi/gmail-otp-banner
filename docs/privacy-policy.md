# Privacy Policy

Effective date: May 26, 2026

## Overview

Gmail OTP Banner is an open-source Chrome extension for local installation. It is not currently available on the Chrome Web Store.

Users download the source code, configure their own Google Cloud project, create their own OAuth Client ID, and install the extension manually using Chrome Developer Mode.

## Data Processing

Gmail OTP Banner connects to Gmail using Google OAuth and requests Gmail read-only access so it can detect verification codes.

The extension processes Gmail data locally in the browser. It only scans recent Gmail messages for OTP detection, and it ignores messages received before the account was connected.

## Data Not Collected Or Sold

Gmail OTP Banner does not sell user data.

Gmail OTP Banner does not send Gmail content to external servers.

Gmail OTP Banner does not use Gmail content for advertising.

Gmail OTP Banner does not store full email bodies.

## Local Storage

The extension may store small local data required for functionality, such as connected account information, recent message IDs, ignored message IDs, and the latest detected OTP.

This information is stored locally in the browser through Chrome extension storage.

## Google OAuth

Each user must configure their own Google Cloud project and OAuth Client ID. Google sign-in is handled by Google, and Gmail OTP Banner never asks for a Gmail password.
