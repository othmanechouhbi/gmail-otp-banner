# Support

Thank you for using Gmail OTP Banner.

## Common Questions

### Why does the extension need Gmail readonly access?

The extension needs Gmail readonly access to search recent Gmail messages for verification-code emails. It uses the minimal Gmail scope required for this feature and does not request send, modify, delete, or compose permissions.

### Does the extension read all my emails?

No. The extension searches recent messages using Gmail search queries focused on verification terms. It scans message content locally only when needed to detect a code.

### Are email contents stored?

No full email body is stored remotely. The extension stores only minimal local operational data such as connected account email, latest OTP, message IDs, and timestamps.

### Why did no banner appear after connecting?

For safety, Gmail OTP Banner does not show old codes that arrived before the account was connected. Send or receive a fresh verification email after connecting the account.

### How do I disconnect an account?

Open the extension popup and click Disconnect next to the account.
